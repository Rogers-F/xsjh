Go 里怎么用 channel 实现一个并发安全的计数器？给个例子。
星
星算助手

可以用一个 goroutine 独占状态、通过 channel 串行化所有操作，避免显式加锁。核心思路：

  • 用一个 ops channel 接收读/写请求；
  • 单个 goroutine 循环处理，天然串行、无数据竞争；
  • 读请求带一个回传 channel 拿结果。
go复制
func counter(ops chan func(int) int) { n := 0 for op := range ops { n = op(n) // 串行执行，安全 } }

这样所有状态变更都在同一个 goroutine 内完成，n 永远不会被并发访问
